# 使用 HPSILab

HPSILab 提供面向股票和期权研究的远程 MCP 工具。用户提出个股分析、隐含波动率、期权压力、蒙特卡洛预测、AI 方向信号、资金曲线、交易前风险检查或研究报告需求时，使用下列工具。

## 工作原则

- 做个股全景判断时，优先调用 `analyze_stock`。它会把关键量化维度放在同一上下文中，适合作为后续深挖的起点。
- 只有在用户明确需要某个维度的更多细节时，再调用对应的专项工具。
- 部分 Pro 工具采用 x402 按次计费。调用前应让用户知道可能产生费用；如果服务返回 x402 支付相关错误，应如实说明需要完成付款或配置支付能力，不要伪造结果。
- 不要凭记忆猜测非美股代码格式。遇到港股、A 股、日股或其他市场代码时，先向用户确认，或依据 HPSILab 返回的格式要求处理。
- 输出应区分事实数据、模型预测与推断，并提醒用户量化分析不构成投资建议。

## 工具

### `analyze_stock`

对指定股票做一次全景量化分析，综合价格、隐含波动率、期权压力、Max Pain、蒙特卡洛区间和 AI 方向信号。个股综合分析应优先使用此工具。

### `get_iv_radar`

查看隐含波动率雷达，包括 IV 水平、期限结构、偏斜或曲面特征。适合判断期权定价是否昂贵、波动率风险集中在哪些期限或执行价。

### `get_option_pressure`

分析期权持仓与交易形成的压力，重点关注 Call/Put 分布、关键执行价、Gamma 或对冲压力，以及 Max Pain。适合研究到期日前后的潜在价格吸引区和压力位。

### `get_monte_carlo`

运行蒙特卡洛模拟，给出未来价格分布、概率区间和情景结果。它描述的是模型分布而非确定目标价，解读时应说明期限、概率和假设。

### `get_ai_prediction`

获取 AI 模型的方向性预测、置信度和相关信号。该结果属于模型判断，应与市场数据和风险指标交叉验证，不能当作确定性交易指令。

### `get_equity_curves`

获取策略或模型的资金曲线及表现指标，用于观察累计收益、回撤和稳定性。比较策略时应同时关注收益与风险，避免只看最终收益率。

### `get_pretrade_risk_scan`

执行交易前风险扫描，识别波动率、流动性、事件、仓位或其他潜在风险。适合在用户准备执行交易前做最后检查；发现高风险项时应明确列出。

### `generate_stock_research_report`

生成指定股票的完整研究报告，汇总多个分析维度并形成结构化结论。这是 Pro 工具，可能触发 x402 按次付费；若支付失败，向用户说明原因和后续处理方式。

### `generate_stock_images`

生成股票研究相关的图表或图片资产，适合报告配图和可视化展示。这是 Pro 工具，可能触发 x402 按次付费；不要在失败时声称图片已经生成。
